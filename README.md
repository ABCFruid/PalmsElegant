# Palm's Elegant

A lightweight utility mod for creating, managing, and snapping to custom viewing angles.

Palm's Elegant allows players to place named angle markers anywhere in their field of view and quickly snap back to them with precision. Whether you're aligning structures, maintaining consistent angles, or working with Mirror Pylons, Palm's Elegant provides a simple and unobtrusive solution.

## Features

* Create unlimited custom angle markers
* Name and color each marker
* Precise yaw and pitch storage
* Visual world-space markers
* Adjustable marker and text scaling
* Adjustable snap distance
* Adjustable snap delay
* Adjustable snap lock duration
* Persistent angle sets
* Per-world angle storage
* Per-server angle storage
* Remembers the last selected angle set
* Always-visible markers that render through blocks
* Lightweight and client-side only

## Use Cases

Palm's Elegant was originally designed as a utility tool for Mirror Pylon placement and alignment, allowing players to create precise, reusable angle references and instantly snap back to them when needed.

## Controls

Open the Palm's Elegant menu using the configured keybind in Minecraft's Controls menu.

From the menu you can:

* Create angle sets
* Add, edit, and delete angles
* Customize marker colors
* Configure snapping behavior
* Adjust marker and text size

## Storage

Palm's Elegant automatically stores data separately for:

* Singleplayer worlds
* Multiplayer servers

This allows every world and server to maintain its own independent collection of angle sets and markers.

## Requirements

* Minecraft 1.21.6
* Fabric Loader
* Fabric API

## Inspiration

Palm's Elegant was inspired by the functionality of AngleSnap and the user interface design philosophy of Xaero's Minimap.

[AngleSnap](https://modrinth.com/mod/anglesnap)

[Xaero's Minimap](https://modrinth.com/mod/xaeros-minimap)

While Palm's Elegant was developed independently, both projects served as inspiration during the design process.
